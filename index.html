HakimSec - WordPress Vulnerability Scanner

Built by Hakim, this scanner performs full analysis on WordPress sites including:

  • Plugin & Theme enumeration
  • Login brute-force (rockyou.txt)
  • CVE detection & CVSS scoring
  • Export to PDF, HTML, Burp JSON

How to Use HakimSec

  1. Clone the repo:
    git clone https://github.com/hakim2345/hakimsec
  2. Install dependencies:
    pip install -r requirements.txt
  3. Scan a target:
    python3 hakimsec.py https://example.com
  4. Reports:
    Saved as report.html, report.json, report.pdf
